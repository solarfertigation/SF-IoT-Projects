# Setup Angular Env on AWS Server 
> Install NodeJS
```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
. ~/.nvm/nvm.sh
nvm install --lts
```

> Install GIT
```
sudo yum install git -y
```

> Install Nginx
```
sudo amazon-linux-extras install nginx1
```

> Install Angular
```
npm install -g @angular/cli
```
